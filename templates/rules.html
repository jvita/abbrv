Rules

Rules can be used to modify text prior to translation into shorthand glyphs. They are applied using find-and-replace operations with regular expressions. If you don't want to learn how to write regular expressions (which can be a bit of a pain), try asking your friendly neighborhood LLM to write them for you! The site regex101 can be useful for making sure that your regular expression is working properly.
