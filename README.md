# abbrv
A framework for designing and using shorthand fonts.

# Getting started

Install the package
```bash
git clone https://github.com/jvita/abbrv.git
cd abbrv
pip install .

```

Start the application:
```
flask run
```

Open http://127.0.0.1:5000 in your browser.

Navigate to the "help" page.
